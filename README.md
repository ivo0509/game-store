# Game Store App

Game Store App is a full-stack digital game store where users can browse games, add games to cart, purchase games, manage their own game library, add games to wishlist and leave reviews.

The project is built with Next.js, React, TypeScript, Tailwind CSS, PostgreSQL, Drizzle ORM and Expo React Native.

## Features

### Visitors
- View home page
- Browse games
- Search games by title, genre or platform
- View game details
- Register and login

### Users
- Manage profile
- Add games to cart
- Remove games from cart
- Complete checkout
- View purchased games in library
- Add games to wishlist
- Leave reviews for purchased games
- Transactions by users to publishers (Add to Card / Refund)  are also executed from the mobile app to the web app


### Publishers
- Add new games
- Edit own games
- Manage game availability
- View sales
- View reviews

### Admins
- Manage users
- Manage games
- Manage orders
- Manage reviews
